# 🌴 GoaGenie — Your AI-Powered Smart Goa Travel Companion

GoaGenie is a premium, all-in-one travel platform designed to transform how you experience Goa. Using state-of-the-art AI, real-time data, and a minimalist modern design, it helps travelers plan, navigate, and save their perfect Goa journey.

## 🌟 Overview

- **7+ Major Core Features** ranging from AI itineraries to real-time voice translation.
- **4 External APIs** seamlessly integrated for maps, weather, AI, and backend services.
- **Full-Stack Architecture** utilizing React (Vite) on the frontend and Node.js/Express on the backend.
- **Serverless Database** powered by Firebase Realtime Database and Authentication.

## ✨ 8 Major Features

1. **🤖 AI-Powered Itineraries** — Personalized day-wise plans generated by **Groq (Llama 3.3)** based on your group type, interests, and budget.
2. **🗣️ Real-time Voice Translator** — Speak in English or Hindi and get instant translations and audio playback in local Konkani.
3. **💾 Persistent Trip Management** — Save your favorite itineraries to your profile via **Firebase**, and manage them anytime in your "My Trips" dashboard.
4. **🚗 Interactive Map Navigation** — Integrated **Google Maps** with geolocation to track your location, calculate distances, and open deep links for driving directions.
5. **⛅ Smart Weather Intelligence** — Live forecasts and proactive travel warnings (rain/storm alerts) powered by **OpenWeather API**.
6. **💬 Smart Travel Chatbot** — A dedicated AI assistant to answer your questions about Goa in real-time.
7. **🎯 Curated Spot Explorer** — Browse curated shacks, beaches, and hidden gems with popup modals and rich data.
8. **👥 Live Traveler Statistics** — Real-time tracking of actual registered users via Firebase.

## 🛠️ Tech Stack & Architecture

### Frontend (Client)
- **Framework**: React.js (Vite)
- **Styling**: Vanilla CSS (Minimalist Flat "Ocean" Design, Custom Animations)
- **Routing**: React Router DOM
- **Speech API**: Native Web Speech API (`SpeechRecognition` & `speechSynthesis`)

### Backend (Server)
- **Runtime**: Node.js
- **Framework**: Express.js
- **Middleware**: CORS, Express JSON parser

### Database & Authentication
- **Provider**: Firebase (BaaS)
- **Auth**: Email/Password & Google OAuth
- **Database**: Firebase Realtime Database

### 🔌 4 External APIs Integrated
1. **Groq API (`llama-3.3-70b-versatile`)**: Powers the AI Itinerary generation, Chatbot, and Voice Translator.
2. **Google Maps JavaScript API**: Powers the interactive maps, geolocation, and distance calculations.
3. **OpenWeather API**: Fetches real-time weather data and 5-day forecasts for Goa.
4. **Firebase API**: Handles secure user authentication and database management.

## 🚀 How to Run the Code

### Prerequisites
- Node.js (v18+)
- npm (Node Package Manager)
- A Firebase Project (with Auth and Realtime DB enabled)
- API Keys for Google Maps, Groq, and OpenWeather

### 1. Clone the Repository
```bash
git clone https://github.com/jitsinghamahapatra/GoaGenie.git
cd GoaGenie
```

### 2. Configure Environment Variables
Create a `.env` file in the **root directory** and add your keys:

```env
# Server Configuration
PORT=5000
GROQ_API_KEY=your_groq_api_key
OPENWEATHER_API_KEY=your_openweather_api_key

# Client Configuration
VITE_API_URL=http://localhost:5000
VITE_FIREBASE_API_KEY=your_firebase_key
VITE_FIREBASE_AUTH_DOMAIN=your_firebase_auth_domain
VITE_FIREBASE_DB_URL=your_firebase_db_url
VITE_FIREBASE_PROJECT_ID=your_firebase_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_firebase_storage_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=your_firebase_sender_id
VITE_FIREBASE_APP_ID=your_firebase_app_id
VITE_GOOGLE_MAPS_API_KEY=your_maps_key
```

### 3. Install Dependencies
We use a root package.json to manage both client and server installations simultaneously.
```bash
npm install
```

### 4. Start the Application
Run both the frontend and backend concurrently with one command:
```bash
npm run dev
```
- **Backend** will run on `http://localhost:5000`
- **Frontend** will run on `http://localhost:5173` (Open this in your browser)
