# 🌴 GoaGenie — Your AI-Powered Smart Goa Travel Companion

GoaGenie is a premium, all-in-one travel platform designed to transform how you experience Goa. Using state-of-the-art AI, real-time data, and a minimalist modern design, it helps travelers plan, navigate, and save their perfect Goa journey.

## ✨ Key Features

- 🤖 **AI-Powered Itineraries** — Personalized day-wise plans generated by **Groq (Llama 3.3)** based on your group type, interests, and budget.
- 💾 **Persistent Trip Management** — Save your favorite itineraries to your profile via **Firebase**, and manage them anytime in your "My Trips" dashboard.
- 🚗 **Real-Time Map Navigation** — Integrated **Google Maps** with geolocation to track your location and get driving directions to any spot in your plan.
- ⛅ **Smart Weather Intelligence** — Live forecasts and proactive travel warnings (rain/storm alerts) powered by **OpenWeather API**.
- 💬 **Smart Travel Chatbot** — A dedicated AI assistant to answer your questions about Goa in real-time.
- 🎯 **Interactive Spot Explorer** — Browse curated shacks, beaches, and hidden gems with a "Track Location" feature that syncs directly with the map.
- 🌊 **Minimalist Modern UI** — A clean, flat "Ocean" aesthetic with immersive parallax water wave animations.

## 🛠️ Tech Stack

| Layer | Technology |
|-------|------------|
| **Frontend** | React.js (Vite), Vanilla CSS (Minimalist Flat Design) |
| **Backend** | Node.js + Express |
| **Database/Auth**| Firebase (Realtime DB & Authentication) |
| **AI Engine** | Groq API (llama-3.3-70b-versatile) |
| **Maps** | Google Maps JavaScript API (Directions & Geolocation) |
| **Weather** | OpenWeather API |

## 🚀 Getting Started

### Prerequisites
- Node.js v18+
- npm
- Firebase Project
- Google Maps API Key
- Groq API Key

### Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/jitsinghamahapatra/GoaGenie.git
   cd GoaGenie
   ```

2. **Install all dependencies**
   ```bash
   npm install        # Installs root, client, and server dependencies
   ```

3. **Run EVERYTHING with ONE command**
   ```bash
   npm run dev
   ```
   *Backend will run on `localhost:5000`*  
   *Frontend will run on `localhost:5173`*

## 🔑 Environment Variables

**`.env` (Root Directory)**
```env
# Server Configuration
PORT=5000
GROQ_API_KEY=your_groq_api_key
RAPIDAPI_KEY=your_rapidapi_key
OPENWEATHER_API_KEY=your_openweather_api_key

# Client Configuration
VITE_API_URL=http://localhost:5000
VITE_FIREBASE_API_KEY=your_firebase_key
VITE_FIREBASE_AUTH_DOMAIN=your_firebase_auth_domain
VITE_FIREBASE_DB_URL=your_firebase_db_url
VITE_FIREBASE_PROJECT_ID=your_firebase_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_firebase_storage_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=your_firebase_sender_id
VITE_FIREBASE_APP_ID=your_firebase_app_id
VITE_GOOGLE_MAPS_API_KEY=your_maps_key
```
